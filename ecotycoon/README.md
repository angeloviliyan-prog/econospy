# EcoTycoon: Claim Your Territory 🎮

**Интерактивна образователна игра по икономика на предприятието за 11–12 клас**

---

## 📖 Описание

EcoTycoon е телевизионно шоу-стил игра, където ученици завладяват територии на карта на България чрез интеллектуални дуели с въпроси по икономика на предприятието.

---

## 🚀 Стартиране

### Предварителни изисквания
- Node.js 14+ ([Изтегляне](https://nodejs.org))

### Linux/Mac
```bash
cd ecotycoon
bash setup.sh
npm start
```

### Windows
```bash
cd ecotycoon
setup.bat
npm start
```

### Отворете в браузър
- **Учител (управление)**: http://localhost:3000/teacher.html
- **Ученик (игра)**: http://localhost:3000/student.html
- **Начало**: http://localhost:3000

---

## 🎮 Как се играе

### Учител (Host)
1. Кликни **"Създай Игра"** → Получи код (напр. `ABC123`)
2. Сподели кода с учениците
3. Всеки ученик влиза с код и име
4. Кликни **"Начало Игра"** → Всеки ученик получава регион
5. Системата показва карта, класация, икономика в реално време

### Ученик (Player)
1. Отворе http://localhost:3000/student.html
2. Въведи **име** и **код на игра**
3. Кликни на съседен регион → Дуел!
4. **60 сек** да отговориш на въпроси
5. Правилен отговор = +1 точка
6. Повече точки = Завладяш региона
7. Печали = +200 капитал

## 📋 Теми на въпросите

| Регион | Тема | Доход |
|--------|------|-------|
| 🔵 **NW** (Северозапад) | Разходи и приходи | 100 |
| 🔵 **NC** (Северен център) | Ценообразуване | 200 |
| 🔵 **NE** (Североизток) | Разходи и себестойност | 150 |
| 🔴 **SW** (Югозапад) | Ресурси | 120 |
| 🔴 **SC** (Южен център) | Персонал | 130 |
| 🔴 **SE** (Югоизток) | Капитал и инвестиции | 180 |

**Всеки регион има 6 въпроса (лесно → трудно)**

---

## 💰 Икономика

- **Начален капитал**: 1000 BGN
- **Пасивен доход**: Всеки 10 сек от твоите региони
- **Награда за победа**: +200 капитал
- **Инвестиция**: -300 капитал → +50 доход

Икономиката се синхронизира в реално време!

---

## ⚔️ Дуел (60 сек)

1. Escolha съседен регион
2. Системата показва **въпрос**
3. **2-3 опции** за отговор
4. Всеки отговор на раунд дава точки
5. Повече точки = Победител
6. При равенство = Sudden Death

## 🛠️ Технологии

- **Backend**: Node.js + Express

- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Дизайн**: Cinematic тъмен стил
